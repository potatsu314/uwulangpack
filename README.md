# uwulangpack
Tool that transforms translation texts from client.jar and mod.jar and resourcepack.zip to uwuified version (only supports english)!

The code is pretty trash, but it works ig.

## Usage
Basic usage :3
```
./uwuify.py -lksc --texts --pack-mcmeta pack.mcmeta --pack-png pack.png --input client.jar mod1.jar mod2.jar --output uwulang.zip
```

Or just get some help :D
```
./uwuify.py --help
```
