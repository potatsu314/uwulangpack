# uwulangpack

Tool that transforms translation texts from client.jar and mod.jar and resourcepack.zip to uwuified version!

The code is pretty trash, but it works ig.

## Usage

Basic usage :3

```
./uwuify.py -lksc --texts --pack-mcmeta pack.mcmeta --pack-png pack.png --language en_us --asset-index 30.json --input client.jar mod1.jar mod2.jar .minecraft/assets --output uwulang.zip
```

Or just get some help :D

```
./uwuify.py --help
```
